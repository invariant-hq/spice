Search the web for current or external information beyond the model cutoff.

Use this tool when the answer depends on recent information, current
documentation, prices, releases, news, or facts that may have changed. For
latest, current, or recent topics, take the current date from the session
context and put that year in the query — asked for "latest X" in 2026,
search "X 2026", not the year your training data suggests. Use domain
filters when the user or task names a source family; broaden them if they
leave too few results. Use web_fetch to inspect a specific result when exact
page content matters.

When search results shaped the answer, end your message with the source
URLs so the user can check them. Never present a searched fact without its
source.
